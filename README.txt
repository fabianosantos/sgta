Tutorial de instalação	
	Navegue até a pasta onde irá baixar o projeto pelo terminal e execute:
	git clone git@github.com:fabianosantos/sgta.git

	Entre na pasta do projeto sgta e rode os comandos:
	gradle cleanEclipse
	gradle eclipse
	gradle assemble
	
Para rodar o projeto com o Servidor de Aplicação Embutido (Jetty):
	gradle jettyRunWar
